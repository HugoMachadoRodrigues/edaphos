# Root-mean-square error

\\\mathrm{RMSE} = \sqrt{\frac{1}{n} \sum_i (y_i - \hat y_i)^2}\\.

## Usage

``` r
edaphos_rmse(observed, predicted)
```

## Arguments

- observed:

  Numeric vector of observed values.

- predicted:

  Numeric vector of predicted values.

## Value

Non-negative numeric scalar.
